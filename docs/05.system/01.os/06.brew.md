## brew命令
```bash
安装软件
brew install git

brew提升权限
sudo chown root /usr/local/bin/brew

卸载软件
brew uninstall wget

查询软件
brew search /wge*/

列出已安装的软件
brew list

更新brew
brew update

显示软件信息
brew info

显示包依赖
brew deps
```
